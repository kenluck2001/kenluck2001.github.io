Cryptography Research: This is a collection of implementation of several NIST cryptographic standards such as ciphers, key generation schemes, demonstrated a novel attack on the Picnic quantum signature scheme and others, 2023.

Developed an generic authenticated scheme for converting any cipher to an authenticated cipher known as Tortoise [ paper, source code].

Differential Privacy Research: This work covers the implementation of Negative Database that saves the complement of records using a hard-to-reverse 3-SAT formulation, a large-scale privacy-aware (locally-private) frequency aggregation of event telemetry, and others, 2023.

Distributed Systems Research: This is a collection of algorithms that I implemented as part of foundational research into Distributed Systems, 2021 - 2023.

Anomaly | anomalyMulti: wrote a modular library for real-time anomaly detection for univariate and multivariate data streams, 2017 - 2020.

pySmooth: wrote a unique time series library in Python that consists of Kalman filters (discrete, extended, and unscented), online ARIMA (novel idea as a personal invention), and time difference model, 2017.
